2. 行级标签：是行内一块内容

(1) 图片 img

  1. 图片的高度和宽度同时指定，可能会产生变形
  2. 图片引用网络资源，src可以写网络路径
  3. 图片引用本地资源，src根据路径编写（上楼下楼）

(2) 超链接a

  1. 超链接跳转定义在href --> html ref引用
  2. 跳转到外站资源 --> 淘宝，网址必须要以http(或https) 开口
  3. 跳转到本站资源，地址格式参考 img的src
  4. 超链接可以实现页面内导航，导航到错点位置 指定的id（锚点）
